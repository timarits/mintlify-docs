### API Endpoints

<AccordionGroup>
  <Accordion title="User API">
    - `POST /v1/projects` - Create projects
    - `POST /v1/integrations/{partner}/verify` - Verify integrations
    - `POST /webhooks/{partner}` - Partner event webhooks
    - `POST /v1/usage/burn` - Burn credits with optional voucher
    - `GET /v1/wallet` - Get wallet balance
    - `POST /v1/vouchers` - Create sponsored vouchers
  </Accordion>
  
  <Accordion title="Partner API">
    - `GET /v1/partners/me` - Partner profile
    - `GET /v1/partners/reports/summary` - Usage reports
    - `GET /v1/partners/invoices` - Invoice management
    - `GET /v1/partners/budgets` - Budget monitoring
  </Accordion>
  
  <Accordion title="Admin API">
    - `POST /v1/billing/reconcile` - Monthly reconciliation
    - `POST /v1/partners/{slug}/budgets` - Budget management
    - `POST /v1/partners/{slug}/rate` - Rate card configuration
    - `POST /v1/partners/{slug}/keys` - API key management
  </Accordion>
</AccordionGroup>
